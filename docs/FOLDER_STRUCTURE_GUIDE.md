# Project Folder Structure and File Purpose

This document lists the current folder structure of `Alumni-Connect-DEP-` and explains what each file does.

## Root

- `.gitignore` - Git ignore rules for the whole repository.
- `LICENSE` - Project license text.
- `README.md` - Top-level project overview and setup guidance.

## docs/

- `docs/API_docs.md` - API endpoint documentation and request/response notes.
- `docs/BACKEND_SETUP.md` - Backend installation, environment, and DB setup steps.
- `docs/CONSOLE_DEBUGGING.md` - Debug notes for client/browser console issues.
- `docs/ER_diagram.png` - Entity relationship diagram for database tables.
- `docs/features.md` - Feature list and module summary.
- `docs/FIX_REGISTRATION.md` - Registration bug fixes and known workarounds.
- `docs/FOLDER_STRUCTURE_GUIDE.md` - This file.
- `docs/LOGIN_PERFORMANCE.md` - Login/session performance notes and tuning.
- `docs/QUICK_START.md` - Fast start guide for local development.
- `docs/REGISTRATION_TROUBLESHOOT.md` - Registration troubleshooting checklist.
- `docs/ROLE_BASED_AUTH_SETUP.md` - Role-based auth and approval flow setup.
- `docs/SESSION_HANDLING.md` - Session lifecycle behavior and handling notes.
- `docs/SESSION_SYNC_FIX.md` - Fix notes for session sync and reload behavior.
- `docs/SUPABASE_APPROVAL_SETUP.md` - Supabase approval/RLS setup instructions.
- `docs/WEBSITE_WORKING.md` - Status and behavior notes for website flows.

## server/

- `server/.env` - Backend environment variables (Supabase keys, secrets, runtime config).
- `server/package-lock.json` - Backend dependency lockfile.
- `server/package.json` - Backend package metadata and npm scripts.
- `server/server.js` - Node server entry point that starts the backend app.
- `server/supabase_otps_table.sql` - SQL to create/manage OTP-related table structure.
- `server/supabase_profiles_backfill.sql` - SQL to backfill missing profile email and normalize approval fields.

### server/src/

- `server/src/app.js` - Express app composition (middleware, routes, error pipeline).

### server/src/config/

- `server/src/config/db.js` - Database/Supabase connection setup helpers.
- `server/src/config/multer.js` - File upload middleware configuration.

### server/src/controllers/

- `server/src/controllers/authController.js` - Auth handlers (signup/login flows, OTP/reset, profile creation logic).
- `server/src/controllers/eventController.js` - Event API request handlers.
- `server/src/controllers/jobController.js` - Job API request handlers.
- `server/src/controllers/postController.js` - Post/feed API request handlers.
- `server/src/controllers/userController.js` - User/profile API request handlers.

### server/src/middleware/

- `server/src/middleware/authMiddleware.js` - Route protection and authentication checks.
- `server/src/middleware/errorMiddleware.js` - Centralized error response middleware.

### server/src/models/

- `server/src/models/eventModel.js` - Event data access/model helpers.
- `server/src/models/jobModel.js` - Job data access/model helpers.
- `server/src/models/postModel.js` - Post data access/model helpers.
- `server/src/models/userModel.js` - User data access/model helpers.

### server/src/routes/

- `server/src/routes/authRoutes.js` - URL routes for auth endpoints.
- `server/src/routes/eventRoutes.js` - URL routes for event endpoints.
- `server/src/routes/jobRoutes.js` - URL routes for job endpoints.
- `server/src/routes/postRoutes.js` - URL routes for post/feed endpoints.
- `server/src/routes/userRoutes.js` - URL routes for user/profile endpoints.

### server/src/services/

- `server/src/services/recommendationService.js` - Recommendation/business logic utilities.

### server/src/utils/

- `server/src/utils/jwt.js` - JWT generation/verification helper utilities.

## client/

- `client/.env` - Frontend environment variables (API base URL, Supabase public config).
- `client/.gitignore` - Git ignore rules specific to frontend workspace.
- `client/eslint.config.js` - ESLint configuration for frontend code quality rules.
- `client/index.html` - Vite HTML entry template.
- `client/package-lock.json` - Frontend dependency lockfile.
- `client/package.json` - Frontend package metadata and npm scripts.
- `client/postcss.config.js` - PostCSS plugin pipeline configuration.
- `client/README.md` - Frontend-specific usage notes.
- `client/SETUP.md` - Frontend setup guide.
- `client/tailwind.config.js` - Tailwind theme/content configuration.
